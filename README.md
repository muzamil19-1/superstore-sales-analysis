# Superstore Sales Analysis

## Project Overview
This project analyzes retail sales data to identify trends in sales, profit, and regional performance.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
The dataset contains retail transaction data including sales, profit, product category, and region.

## Analysis Performed
- Data cleaning
- Sales analysis by category
- Regional sales performance
- Top selling products
- Monthly sales trends

## Key Insights
- Technology category generates highest revenue
- West region has highest sales
- Some products generate high sales but low profit
